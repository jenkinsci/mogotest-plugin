Before you can trigger a test run through Jenkins you must first configure a test plan, either through the Mogotest web site or through our API. If you use the API, the test plan ID is the normal one as outlined in the API docs. If you configure through the web site, the test plan ID is part of the URL. E.g., in the URL https://mogotest.com/sites/www-quirksmode-org/groups/s5qtxk/tests/latest, the test plan ID is s5qtxk.
